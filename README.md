# Web Application Attack Surface Mapping Tool

## Description

This project is an automated tool for identifying and mapping the software-based
attack surface of a web application. It focuses on publicly accessible entry points
such as web ports, login pages, APIs, forms, exposed directories, and technologies used.

## Scope

- Web application attack surfaces only
- No exploitation or destructive testing
- Ethical and permission-based usage

## Features (Planned)

- Web port discovery
- Endpoint and directory discovery
- Login and admin page detection
- API endpoint detection
- Web form and file upload detection
- Technology fingerprinting (WhatWeb)
- Risk classification
- HTML/PDF reporting

## Platform

- macOS
- Python 3

## Ethical Notice

This tool is intended for educational and authorized security testing only.
